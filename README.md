Easy-PIL
========

PIL made easy.

Installation
------------

    git clone git@github.com:jpanganiban/easy-pil.git

    python setup.py install/develop

Running Tests
-------------

    python setup.py test


Usage
-----

Inherit PILImage object

    from easypil import PILImage

    class MyPILImage(PILImage):
      # Do awesome stuff
      pass
